CSS Color

Setting Color Property The color property defines the text color (foreground color in general) of an element. For instance, the color property specified in the body selector defines the default text color for the whole page. Let's try out the following example to see how it works: body { color: #ff5722; } Defining Color Values Colors in CSS most often specified in the following formats: ● a color keyword - like "red", "green", "blue", "transparent", etc. ● a HEX value - like "#ff0000", "#00ff00", etc. ● an RGB value - like "rgb(255, 0, 0)"
