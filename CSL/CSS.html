Universal Selector

The universal selector, denoted by an asterisk (*), matches every single element on the page. The universal selector may be omitted if other conditions exist on the element. CSS Selectors

A CSS selector is a pattern to match the elements on a web page. The style rules associated with that selector will be applied to the elements that match the selector pattern. Selectors are one of the most important aspects of CSS as they allow you to target specific elements on your web page in various ways so that they can be styled. Several types of selectors are available in CSS, let's take a closer look at them:

    An Element type selector matches all instance of the element in the document with the corresponding
  • element type name. Let's try out an example to see how it actually works: p { color: blue; }
    • The style rules inside the p selector will be applied on every element (or paragraph) in the document and color it blue, regardless of their position in the document tree
